A.9. Tools

507. NCI Agency exploited the capability to semi-automatically generate code to create tools to help validate the XML files in support of Interoperability Testing. In particular, the SMACQ/O-ANT tool suite is available that can be used to monitor the information exchange and to report on its compliance to the relevant Standards.

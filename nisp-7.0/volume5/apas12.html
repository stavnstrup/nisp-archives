A.12. Future Plans

510. Work on the STF will continue with capturing further the missing aspects of current STF layers and adding the Design Rules and Methodology for additional layers including the XML Schemas to support it. The following is a planned list of items to work on:

511.

  • Data Element Dictionary and Message Structure for XML-Based information exchanges

  • Message Structure for Variable-length Bit-based information exchanges

  • Security Cross-domain Layer
